Мой небольшой проект в котором я использовал ООП и php.

Целью данного проекта было разобраться в ООП.

### Как запустить:
1. Заходим в папку public/
(У вас должен быть установлен php)
2. И вводим команду в консоль php -S localhost:8001
(Я запускал на своем arch linux)

### Немного о структуре
- В папке **data** находится **"база данных" в формате .json**
- В папке **public** все **страницы, стили и tailwind css** сайта
- В папке **src** находиться **логика (ООП)**
- В **vendor** находиться **композер для ООП**
